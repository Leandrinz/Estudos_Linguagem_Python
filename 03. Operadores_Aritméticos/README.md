# 🐍 Operadores Aritméticos (Python)

---

## 📌 1. Operadores básicos

Python possui diversos operadores aritméticos para realizar cálculos matemáticos.

### ➕ Adição

```python
+
```

Soma dois valores.

---

### ➖ Subtração

```python
-
```

Subtrai um valor do outro.

---

### ✖️ Multiplicação

```python
*
```

Multiplica dois valores.

---

### ➗ Divisão

```python
/
```

Divide e retorna um número decimal (float).

---

### 🔼 Potência

```python
**
```

Eleva um número a uma potência.

Exemplo:

```python
2 ** 3  # 8
```

---

### 🔢 Divisão inteira

```python
//
```

Retorna apenas a parte inteira da divisão.

Exemplo:

```python
7 // 2  # 3
```

---

### 📦 Resto da divisão (módulo)

```python
%
```

Retorna o resto da divisão.

Exemplo:

```python
7 % 2  # 1
```

---

### ⚖️ Igualdade

```python
==
```

Compara se dois valores são iguais.

Exemplo:

```python
5 == 5  # True
```

---

## 📌 2. Ordem de precedência

A ordem em que as operações são resolvidas:

```
( ) → ** → * / // % → + -
```

### 🧠 Exemplo:

```python
2 + 3 * 4
```

Resultado:

```
14 (multiplicação vem antes da soma)
```

---

## 🚀 Resumo

* `+` → soma
* `-` → subtração
* `*` → multiplicação
* `/` → divisão normal
* `**` → potência
* `//` → divisão inteira
* `%` → resto
* `==` → comparação
* Ordem: `( ) → ** → * / // % → + -`
